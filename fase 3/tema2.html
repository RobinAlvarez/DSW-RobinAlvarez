Atributos y valores

Estas permiten se usada dentro de las etiquetas para realizar actividades de esta y poder modificar
los elementos de HTML, y se añade al inicio de la etiqueta de HTML, tenemos los siguientes atributos:

Id: este permite la identidad
Class: proporciona la clasificación del elemento.
Style: estilo de cualquier elemento en HTML.
Title: permite llevar a cabo el título del documento HTML.
