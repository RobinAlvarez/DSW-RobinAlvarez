Etiquetas Básicas de HTML

Las etiquetas de HTML son las que indican al navegador que el código siguiente es puro HTML, definido
por su tipo de documento como DOCTYPE, es el principal de raíz también determinamos el script permite
llamar url en la web, head> nos indica la cabeza del documento HTML, title define el título, link vincula
hojas de estilos o iconos, style para seleccionar el estilo realizado por css, meta licencia para indicar
el http, body define el contenido principal del documento entre estas la principales también tenemos la siguientes etiquetas:
Div, Table, Img, Src, Ul ,Ol, B, I, S, U, Main, Span, Br, Details, Summary. Todas deben de estar dentro de <>.

<Div><,Table><,Img>,<Src>,<Ul>,<li>,<ol>,<b>,<i>,<s>,<u>,<main>,<span>,<br>, Más

Atributos y valores

Estas permiten se usada dentro de las etiquetas para realizar actividades de esta y poder modificar
los elementos de HTML, y se añade al inicio de la etiqueta de HTML, tenemos los siguientes atributos:

Id: este permite la identidad
Class: proporciona la clasificación del elemento.
Style: estilo de cualquier elemento en HTML.
Title: permite llevar a cabo el título del documento HTML.
